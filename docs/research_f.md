## 概要

調査・リサーチ系の副業ビジネスアイデア10件について、市場規模、単価相場、参入障壁、マルチエージェントとの相性などを調査した。

## 1. 市場調査レポート作成

### 基本情報
| 項目 | 内容 |
|------|------|
| **市場規模・需要** | ITフリーランス市場規模 約1兆1,849億円（2025年）、コンサル案件は増加傾向 |
| **競合状況** | 大手調査会社（矢野経済、日経BP等）、フリーランスプラットフォーム（ランサーズ等） |
| **想定単価** | 時給2,000〜10,000円、月額60〜170万円（コンサル領域） |
| **必要スキル** | 情報収集力、分析力、PowerPoint/Excel、業界知識 |
| **参入障壁** | **低〜中** |
| **スケーラビリティ** | **高** |
| **マルチエージェント相性** | **高** — データ収集・分析・レポート生成を分担可能 |
| **即金性** | **中** — 案件獲得後1〜2週間で納品 |

### 具体的な営業先・顧客獲得方法
- クラウドソーシング（ランサーズ、クラウドワークス）
- フリーコンサルエージェント（ProConnect、POD等）
- 直接営業（スタートアップ、中小企業の経営企画部門）
- ビジネスマッチングプラットフォーム

### リスク・注意点
- 品質のばらつきが評価に直結
- 機密情報の取り扱いに注意（NDA必須）
- 納期厳守が信頼に直結

## 4. 特許調査

### 基本情報
| 項目 | 内容 |
|------|------|
| **市場規模・需要** | 日本の特許出願 年間30万件超、継続的需要あり |
| **競合状況** | 特許事務所、日本IR、専門調査会社、個人サーチャー |
| **想定単価** | 1案件10〜40万円、業務委託で60%が報酬 |
| **必要スキル** | 特許検索技術、技術文書読解力、J-PlatPat操作 |
| **参入障壁** | **中〜高** — 専門知識と経験が必要 |
| **スケーラビリティ** | **中** |
| **マルチエージェント相性** | **中** — 検索は自動化可能だが分析は人間が必要 |
| **即金性** | **中** — 1案件1〜3週間 |

### 具体的な営業先・顧客獲得方法
- 特許事務所からの下請け
- クラウドワークス（特許調査カテゴリ）
- 製造業の知財部門への直接営業
- 日本IR等の調査会社への登録

### リスク・注意点
- 専門用語・技術理解が必須
- 調査漏れのリスク（訴訟リスクに直結）
- 作業時間の見積もりが難しい

## 5. 法規制調査

### 基本情報
| 項目 | 内容 |
|------|------|
| **市場規模・需要** | 新規事業、海外展開時に需要大、規制産業（金融、医療等）は特に高需要 |
| **競合状況** | 弁護士事務所、リーガルテック（Legalscape等）、パラリーガル |
| **想定単価** | 月額5.5万円〜（リーガルリサーチサービス）、弁護士相談30分5,000円 |
| **必要スキル** | 法律知識、リーガルリサーチツール操作、文書作成力 |
| **参入障壁** | **高** — 弁護士資格なしでは範囲限定 |
| **スケーラビリティ** | **中** |
| **マルチエージェント相性** | **中** — 情報収集は可能だが解釈は専門家が必要 |
| **即金性** | **低〜中** |

### 具体的な営業先・顧客獲得方法
- 法律事務所のリサーチ業務サポート
- スタートアップ（新規事業の適法性確認）
- フリーランス保護法対応コンサル
- 海外展開企業の規制調査支援

### リスク・注意点
- 弁護士法（非弁行為）に抵触しないよう注意
- 法改正への追従が必要
- 誤った情報提供の責任リスク
