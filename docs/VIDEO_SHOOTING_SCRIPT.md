## Case 3: GUIDANCE + Agenticループ（0:55-2:30）★最重要パート

**ここが勝敗を分ける95秒。4つの瞬間を確実に撮る。**

---

### 瞬間1: 止まる（0:55-1:20）25秒

#### 操作

1. `demo_guidance.pdf` をアップロード
2. 「PDFを判定」ボタンをクリック
3. GUIDANCE結果表示 → **2秒の沈黙**
4. 黄色い「要確認」バッジをカーソルで指す

#### セリフ

```
【0:55-1:00】
「では、判断が割れるケース。
 撤去と移設が混在した工事、27万5千円。
 人間でも迷います。」

【1:00-1:05】（アップロード+判定）
「判定します。」

【1:05-1:10】（結果表示後、2秒の沈黙）

「......止まりました。」

【1:10-1:20】（声を落として）
「『要確認』。AIが自分で『判断できない』と認識して、
 自動処理を停止しました。
 これが Stop-first 設計です。」
```

#### 演出
- GUIDANCE表示後の**2秒の完全沈黙**が最重要
- 「止まりました」は低い声でゆっくり
- **ここでスクリーンショット1を取得**（`Win+Shift+S`）

---

### 瞬間2: 聞く — 1回目の質問（1:20-1:40）20秒

#### 操作

1. 「追加情報が必要です」パネルにカーソル移動
2. 不足情報・理由の表示を指し示す
3. 類似事例（紫カード）があれば一瞬触れる
4. プログレス表示を指す（● 質問1 ─ ○ 質問2 ─ ○ 判定）

#### セリフ

```
【1:20-1:30】（パネルを指し示す）
「ここが重要です。
 単に『わかりません』ではない。
 『何の情報が不足しているか』
 『なぜそれが判断に必要か』を具体的に説明しています。」

【1:30-1:35】（類似事例カードを指す ※表示されている場合）
「さらに、似た実務事例を1件提示。
 過去の判断事例から、見落としがちな視点を補います。」

【1:35-1:40】
「では、質問に答えます。今回は『新規購入・増強』を選択。」
```

#### 操作（セリフ後）
5. 「新規購入・設備増強」ボタンをクリック → 画面が Step 2 に遷移

#### 演出
- 不足情報のテキストをカーソルでなぞる
- 類似事例が出ていれば紫カードに**一瞬だけ**触れる（深入りしない）
- **ここでスクリーンショット2を取得**（GUIDANCE全体画面）

---

### 瞬間3: 深く聞く — 2回目の質問（1:40-2:00）20秒

#### 操作

1. Step 2 画面を見せる（プログレス: ● 質問1 ─ ● 質問2 ─ ○ 判定）
2. Step 1の選択済み表示（緑ボックス「新規購入・増強を選択済み」）を指す
3. 詳細質問「既存設備の入替ですか？」を読む
4. 「はい（入替）」ボタンをクリック

#### セリフ

```
【1:40-1:48】（Step 2画面を指し示す）
「1回聞いて終わりではありません。
 AIがさらに踏み込んだ質問をします。
 『既存設備の入替ですか？』
 回答によって、判定結果も変わります。」

【1:48-1:55】
「今回は『はい、入替』を選択。」

【1:55-2:00】（ボタンクリック → 再判定が実行される）
「AIが回答を踏まえて、再判定します。」
```

#### 演出
- プログレス表示の**ドットが進んでいる**ことをカーソルで示す
- 「1回聞いて終わりではありません」を強調
- **ここでスクリーンショット3を取得**（2段階質問画面）

## トラブルシューティング

### Cloud Runが応答しない
1. 1-2分待って再試行（コールドスタート）
2. それでもダメ → ローカル版に切替:
   ```powershell
   cd C:\Users\owner\Desktop\fixed-asset-ashigaru
   streamlit run ui/app_minimal.py
   ```
   UIサイドバーで接続先を `http://localhost:8000` に変更

### GUIDANCE が出ない（demo_guidance.pdfで）
- demo_guidance.pdf の内容が「撤去・移設」を含んでいるか確認
- 含んでいれば必ずGUIDANCEになる（mixed_keywordルール）
- 最悪の場合、JSON入力に切替: `data/demo/demo04_guidance_ambiguous.json`

### 2段階質問が表示されない
- guidance_panel.py がインポートされているか確認
- `_COMPONENTS_AVAILABLE = True` であること
- Session stateをリセット: ブラウザリロード
