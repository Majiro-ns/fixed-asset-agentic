## 3. 魅せるポイント（3つの瞬間）

### 瞬間1: 「止まる」（1:30頃）
- **画面**: GUIDANCE（黄色バッジ）が表示される
- **演出**: 少し間を置いて「止まりました」と強調
- **意図**: AIが断定しない安全設計を印象付ける

### 瞬間2: 「聞く」（1:45頃）
- **画面**: 「追加情報が必要です」パネル + 入力フォーム
- **演出**: 「何が不足しているか、なぜ必要かを説明しています」
- **意図**: 単なるエラーではなく、人間との対話であることを示す

### 瞬間3: 「変わる」（2:15頃）
- **画面**: DIFF表示（Before → After）
- **演出**: 「追加情報をもとに判定が変わりました」
- **意図**: Agenticループの完結を視覚的に示す

## 6. 撮影ツール推奨

### OBS Studio設定

| 項目 | 推奨値 |
|------|--------|
| 出力解像度 | 1920 x 1080 |
| フレームレート | 30fps |
| エンコーダ | x264（CPU）またはNVENC（GPU） |
| ビットレート | 6000 Kbps |
| 出力形式 | MP4 |

#### OBS設定手順
1. 「設定」→「出力」→「出力モード: 詳細」
2. 「録画」タブ → 録画フォーマット: mp4
3. エンコーダ: x264 または NVIDIA NVENC
4. レート制御: CBR、ビットレート: 6000

### マイク設定

- **推奨**: 外部マイク（USB or ヘッドセット）
- **内蔵マイク使用時**: 静かな環境で録音
- **ノイズ抑制**: OBSのフィルタで「ノイズ抑制」を追加
- **音量**: -12dB 〜 -6dB（ピーク時）

### 代替ツール

| ツール | 特徴 |
|--------|------|
| Windows Game Bar | Win+G で起動、簡単だが機能限定 |
| Loom | ブラウザ拡張、クラウド保存、共有が楽 |
| PowerPoint録画 | スライド併用時に便利 |

### 書き出し後

1. **ファイル名**: `fixed-asset-demo-YYYYMMDD.mp4`
2. **アップロード先**: YouTube（限定公開）または Vimeo
3. **Zenn記事に埋め込み**: YouTube URLを記事に貼る
