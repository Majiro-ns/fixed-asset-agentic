## 1. コードレビュー代行

### 概要
他社・個人開発者のコードを第三者視点でレビューし、品質向上を支援するサービス。

### 調査結果

| 項目 | 内容 |
|------|------|
| **市場規模・需要** | 中小企業・スタートアップでシニアエンジニア不在のチームが増加。需要拡大中 |
| **競合状況** | 専門サービスは少ない。Menta、CODEAL等で個人提供が散見 |
| **想定単価** | 時給8,000〜15,000円、1レビュー5,000〜30,000円（規模による） |
| **必要スキル** | 複数言語の深い理解、設計パターン、セキュリティ知識 |
| **参入障壁** | **低** — 実績があれば即開始可能 |
| **スケーラビリティ** | **中** — チェックリスト化で効率化可能 |
| **MA相性** | **高** — 静的解析・パターン検出をエージェントで自動化可能 |
| **即金性** | **高** — 単発案件が多く即金化しやすい |
| **営業先** | Menta、CODEAL、Twitter/X、技術コミュニティ、スタートアップ |
| **リスク** | 責任範囲の明確化必要、レビュー後のバグ発生時の対応 |

### マルチエージェント活用案
- Agent A: 静的解析・コード品質スコア算出
- Agent B: セキュリティ脆弱性パターン検出
- Agent C: ベストプラクティス提案生成
- 人間: 最終レビュー・コミュニケーション

## 2. バグ修正・デバッグサービス

### 概要
既存システムのバグを特定・修正するスポットサービス。

### 調査結果

| 項目 | 内容 |
|------|------|
| **市場規模・需要** | クラウドソーシングで常時1,600件以上の案件（ランサーズ） |
| **競合状況** | ランサーズ、クラウドワークス、ココナラで多数の出品者 |
| **想定単価** | 月15〜20万円（クラウドソーシング）、時給3,000〜8,000円（直接契約） |
| **必要スキル** | デバッグ技法、ログ解析、複数フレームワークの知識 |
| **参入障壁** | **低** — 経験が浅くても小規模案件から開始可能 |
| **スケーラビリティ** | **中** — 案件ごとに異なるため標準化困難 |
| **MA相性** | **高** — エラーログ解析・原因特定をエージェントで効率化 |
| **即金性** | **高** — 短期案件が多い |
| **営業先** | ランサーズ、クラウドワークス、ココナラ、Twitter/X |
| **リスク** | 低単価競争、修正後の追加バグ発生リスク |

### マルチエージェント活用案
- Agent A: エラーログ解析・原因仮説生成
- Agent B: 類似バグパターンDB検索
- Agent C: 修正コード候補生成
- 人間: 最終判断・修正適用
